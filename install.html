eLauncher

App Launcher untuk sistem sekolah.
Install terus pada Windows — tiada perlu Microsoft Store.

⊞ Install untuk Windows

Cara Install

1
Klik butang Install untuk Windows di atas
2
Windows akan buka dialog installer automatik — klik Install
3
Selesai! Cari eLauncher dalam Start Menu
⚠
Jika gagal, pergi Settings → System → For Developers → aktifkan Developer Mode
